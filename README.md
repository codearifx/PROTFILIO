# Personal Portfolio

A modern, responsive, and highly interactive personal portfolio website designed to showcase projects, skills, experience, and certificates.

## 💻 Tech Stack & Languages

- **Language:** JavaScript / JSX
- **Library:** React (v19)
- **Build Tool:** Vite
- **Styling:** Tailwind CSS (v3.4)

## 🎨 CSS & Styling

This project uses **Tailwind CSS** for fully responsive and utility-first styling. It also utilizes custom CSS (`index.css` and `App.css`) for specific animations, gradient text effects, and glassmorphism (glass-card) designs.

## 📦 What's Inside? (Key Dependencies)

The project leverages several powerful libraries to enhance the user experience:
- **framer-motion:** Used for smooth scroll animations, revealing elements, and seamless layout transitions between sections (like the dynamic profile image).
- **react-type-animation:** For the typewriter effect in the Hero section.
- **react-countup:** To animate statistical numbers in the About section.
- **react-icons:** For crisp and scalable icons across the application.
- **react-fast-marquee:** For continuous scrolling bands (e.g., tech stack logos).
- **swiper:** For touch-friendly carousels and sliders (e.g., certificates or projects).
- **clsx & tailwind-merge:** Used securely merge Tailwind classes conditionally.

## 📂 Project Structure & Sections

The portfolio is structured as a Single Page Application (SPA) with smooth scrolling navigation. The key sections included are:

- **Hero:** A welcoming section with a typing animation, parallax effects, and a dynamic shared profile image.
- **About:** Personal introduction, background, and animated statistics.
- **Skills:** A comprehensive list of technical skills and tools.
- **Experience:** Timeline of professional work and internships.
- **Education:** Academic background and achievements.
- **Projects:** Showcase of development projects with descriptions and links.
- **Certificates:** Displays earned certifications and credentials.
- **Contact:** Links to social media profiles and a way to get in touch.

## 🚀 How to Run Locally

1. Install the dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
